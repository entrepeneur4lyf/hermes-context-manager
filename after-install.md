# Hermes Context Manager Installed

- Restart Hermes so the plugin is discovered:
  - `hermes gateway restart`
- Use `/hmc` inside Hermes once the bundled skill is synced.
- Edit `config.yaml` in the installed plugin directory to tune thresholds or manual mode defaults.

This plugin is designed to coexist with Hermes native compaction and reduce the amount of stale noise that reaches it.
